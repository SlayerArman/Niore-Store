## ✨ About

Noire Lifestyle is a clean and modern fashion store website that I built to improve my frontend development skills while creating something that actually looks and feels like a real brand.

Everything was coded manually in **Visual Studio Code** using only:

* HTML
* CSS
* JavaScript

No website builders.

No templates.

Just code.
